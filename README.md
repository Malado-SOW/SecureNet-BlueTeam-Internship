# 🛡️ SecureNet Solutions – Blue Team Internship Project

## 📋 Présentation
Projet de fin d'études – Stage de 6 mois en cybersécurité offensive/défensive.
Simulation d'une infrastructure d'entreprise sécurisée avec supervision SOC.

## 🎯 Objectifs
- [x] Concevoir une architecture réseau segmentée et isolée
- [ ] Déployer un Active Directory sécurisé
- [ ] Implémenter un SIEM Wazuh avec détection avancée
- [ ] Simuler des attaques contrôlées et documenter la réponse à incident
- [ ] Produire une documentation technique professionnelle

## 🚀 Démarrage Rapide
1. Cloner le repository
2. Vérifier les prérequis VirtualBox (voir 01_Documentation_Generale/PREREQUIS.md)
3. Suivre les sprints dans l'ordre chronologique

## 🔐 Sécurité
⚠️ Ce laboratoire contient des configurations de test et des outils d'attaque.
Il doit être exécuté dans un environnement VirtualBox strictement isolé.
Aucune machine ne doit être exposée sur un réseau de production.
